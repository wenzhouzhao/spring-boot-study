## 项目简介
`spring boot study` 是一个用来深度学习并实战 `spring boot` 的项目。


## 开发环境

- **JDK 1.8 +**
- **Maven 3.5 +**
- **IntelliJ IDEA ULTIMATE 2018.2 +** (*使用 IDEA 开发真香，开发前请保证安装 `lombok` 插件*)
- **Mysql 5.7 +**
